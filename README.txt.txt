This is a tool for prediction if an email is going to be opened or not.
The tool also gives interesting insight about the mailbox, like the most frequent words in subject lines in emails you had opened or emails that you have not opened.

This version currently supports only gmail mailbox.
To doenload your own email, go to: https://takeout.google.com/
